# FundPulse — Simple normalized score checker for your mutual fund data

- a basic full stack project helping you to compare mutual funds on the basis of normalized price based score
- link : https://fund-pulse.vercel.app/

# deployment strategy

- I used aiven for cloud database, connected it to the backend hosted on render
- the backend is serving the frontend hosted on vercel

- all of this can be achieved in other ways like AWS EC2, CDNs if we have lot of static files nd Object stores like S3
